weekRelative(currentWeek, n)

Returns an array of two elements representing the week weeks before/after . If is negative, the week - weeks ago is returned; if is positive, the week weeks in the future is returned; and if is 0, is returned. The first element of the returned array is a moment object representing the first day of the new week (Sunday), and the second element is a moment object representing the last day of the new week (Saturday).
